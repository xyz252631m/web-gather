webview app 开发配置

WebView使用了WebKit渲染引擎来显示网页，可以将其视为一个浏览器。

安装和使用步骤如下：

安装java jdk -> 配置环境变量 -> 安装 Android SDK -> 创建模拟器 -> 配置vs2015 -> 项目配置

1.安装java jdk

打开 oracle 官网,下载java jdk，安装即可。

2.配置环境变量

环境变量中新建JAVA_HOME 其值为：jdk的安装路径

添加PATH属性 其值为：%JAVA_HOME%\bin

新建CLASSPATH 值为：.;%JAVA_HOME%\lib

通过cmd运行以下命令：java -version，javac 如果出现返回信息，则设置成功。

3.安装 Android SDK

Android SDK 官网下载地址：http://developer.android.com/sdk/index.html

网速慢打不开，也可以在国内一些镜像站下载，如中科院开源软件镜像站(http://mirrors.opencas.cn/)

运行安装文件，完成后把安装目录下tools文件夹路径设置进PATH环境变量

到安装目录运行SDK Manager，开始什么平台和工具也没有，需要下载

如果速度过慢，可以进入到 Tools -> Options把代理填上。

勾选如下，只选择了安卓4.3版本。

4.创建模拟器

到Android SDK 安装目录运行AVD Manager,点击create按钮，创建模拟器。其中的Target 为安卓版本。

5.配置vs2015

进入工具->选项->Xamarin 指定java的jdk目录，安卓sdk目录

6.项目配置

新建安卓项目，选择WebView App

完成后，右键项目属性，选择安卓版本，此处版本需要有模拟器对应，否则项目无法启动。

如果要在真机测试，请修改此处设置
